# 股票投资管理系统 - 系统架构设计文档

## 1. 系统概述

### 1.1 系统定位
股票投资管理系统是一个集股票数据管理、投资分析、预测模型、持仓管理于一体的综合性投资辅助系统。系统采用B/S架构，提供Web界面进行股票数据分析和投资决策支持。

### 1.2 核心功能
- 股票数据管理（A股全量数据）
- 自选股管理
- K线数据展示（日线、月线）
- 实时行情数据
- 持仓管理（基于券商对账单）
- 预测模型分析
- 智能学习参数
- 新闻资讯集成
- 专家分析报告

## 3. 数据库设计

### 3.1 核心数据表结构

#### 3.1.1 股票基础信息表 (stocks)
```sql
CREATE TABLE stocks (
    code TEXT PRIMARY KEY,      -- 股票代码
    name TEXT NOT NULL,         -- 股票名称
    market TEXT NOT NULL,       -- 市场(sh/sz)
    py TEXT,                    -- 拼音缩写
    watchlist INTEGER DEFAULT 0 -- 是否在自选股
);
```

#### 3.1.2 自选股表 (watchlist)
```sql
CREATE TABLE watchlist (
    code TEXT PRIMARY KEY,      -- 股票代码
    name TEXT NOT NULL,         -- 股票名称
    market TEXT NOT NULL,       -- 市场
    sort_order INTEGER          -- 排序序号
);
```

#### 3.1.3 K线数据表 (kline_daily)
```sql
CREATE TABLE kline_daily (
    id INTEGER PRIMARY KEY,
    code TEXT NOT NULL,         -- 股票代码
    date TEXT NOT NULL,         -- 日期
    open REAL,                  -- 开盘价
    close REAL,                 -- 收盘价
    high REAL,                  -- 最高价
    low REAL                    -- 最低价
);
```

#### 3.1.4 行情数据表 (quotes)
```sql
CREATE TABLE quotes (
    code TEXT PRIMARY KEY,      -- 股票代码
    price REAL,                 -- 当前价格
    change REAL,                -- 涨跌幅
    open REAL,                  -- 开盘价
    high REAL,                  -- 最高价
    low REAL,                   -- 最低价
    pe REAL,                    -- 市盈率
    pb REAL,                    -- 市净率
    dy REAL                     -- 股息率
);
```

#### 3.1.5 持仓数据表 (positions)
```sql
CREATE TABLE positions (
    code TEXT PRIMARY KEY,      -- 股票代码
    name TEXT NOT NULL,         -- 股票名称
    qty INTEGER,                -- 持有数量
    total_cost REAL,            -- 总成本
    avg_cost REAL,              -- 平均成本
    realized_pnl REAL           -- 已实现盈亏
);
```

#### 3.1.6 预测数据表 (daily_predictions)
```sql
CREATE TABLE daily_predictions (
    id INTEGER PRIMARY KEY,
    code TEXT NOT NULL,         -- 股票代码
    date TEXT NOT NULL,         -- 预测日期
    prev_close REAL,            -- 前收盘价
    direction TEXT,             -- 预测方向
    confidence REAL,            -- 置信度
    high REAL,                  -- 预测最高价
    low REAL,                   -- 预测最低价
    advice TEXT,                -- 操作建议
    entry_zone TEXT             -- 入场区域
);
```

### 3.2 数据关系图
```

  ↓
positions (1) ← (N) trades
```

### 6.3 前端核心交互组件

#### 6.3.1 API调用封装层

```javascript
// 位置：bank-stock-system.html 第 1058-1068 行
const API_BASE = (location.hostname==='localhost'||location.hostname==='127.0.0.1')
    ? 'http://localhost:8765' : '';

function hasAPI(){
    return !!API_BASE;
}

async function apiCall(method, path, body){
    if(!hasAPI()){
        alert('请在本地服务器模式下使用(访问 http://localhost:8765 而非直接打开文件)');
        return null;
    }
    try{
        var opts={
            method: method,
            headers: {'Content-Type': 'application/json'}
        };
        if(body) opts.body = JSON.stringify(body);
        var r = await fetch(API_BASE + path, opts);
        return await r.json();
    }catch(e){
        console.error('API error:', e);
        return {success: false, error: e.message};
    }
}
```

#### 6.5.2 添加自选股 — `POST /api/watchlist/add`

```
触发: 搜索选择 or 手动输入 → ajaxAddStock(code,name,market)

1. 前端验证
   if (!code || !name) → alert 提示

2. POST → /api/watchlist/add
   Body: {"code":"600036","name":"招商银行","market":"sh"}

3. 后端 server.py do_POST
   params = json.loads(body)
   code = params["code"], name = params["name"], market = params["market"]

   3a. 读取 data/watchlist.json
   3b. 去重检查: 已存在 → 409 Conflict
   3c. wl["stocks"].append({code, name, market})
   3d. write_json → data/watchlist.json
   3e. 同步 data/system_data.json
   3f. subprocess: python sync_all.py (timeout=120s)
       → 为新股票初始化K线/预测/季节性数据

4. 响应
   { "success": true, "message": "已添加 招商银行(600036)，全模块数据同步完成",
     "watchlist": {...}, "output": "...sync_all输出..." }

5. 前端 onSuccess
   mgmt-status.textContent = result.message
   刷新自选股列表 refreshWatchlistUI()
```
