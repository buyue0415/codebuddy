## 1. 业务需求说明书

### 1.1 业务背景

K线形态是重要的技术分析手段。系统内置33条标准K线形态规则库，并支持用户自定义规则和管理。

### 1.2 核心目标

| 目标 | 说明 |
|------|------|
| 规则列表 | 展示所有K线形态规则（内置+自定义） |
| 规则CRUD | 创建/读取/更新/删除形态规则 |
| K线扫描 | 对指定股票的K线数据应用规则扫描 |
| 匹配结果 | 显示匹配到的K线形态 |

## 3. 业务逻辑

### 3.1 内置规则（33条标准形态）

| 类别 | 规则数 | 示例 |
|------|--------|------|
| 单根形态 | 10 | 十字星、锤子线、射击之星、大阳线、大阴线 |
| 双根形态 | 8 | 看涨吞没、看跌吞没、乌云盖顶、刺透形态 |
| 三根形态 | 10 | 早晨之星、黄昏之星、三只乌鸦、红三兵 |
| 复合形态 | 5 | 头肩顶、双底、旗形、楔形、三角形 |

### 3.2 规则数据结构

```json
{
  "id": 1,
  "name": "早晨之星",
  "description": "下跌趋势末期的反转形态，由长阴线+小实体+长阳线组成",
  "buy_signal": "三重看涨",
  "sell_signal": "",
  "priority": 1
}
```

## 4. 交互流程

```
挂载 → GET /api/v2/pattern-rules → 渲染规则列表表格
  → 编辑: 行内编辑模式 → PUT /api/v2/pattern-rules/{id}
  → 删除: 确认弹窗 → DELETE /api/v2/pattern-rules/{id}
  → 新增: 表单弹窗 → POST /api/v2/pattern-rules
  → 扫描: 选择股票 → GET /api/v2/pattern-rules/scan?code=XXX
```
