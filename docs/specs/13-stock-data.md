## 1. 业务需求说明书

### 1.1 业务背景

投资者需要集中查看所有自选股的实时行情数据，包括价格、涨跌、成交量、市盈率等基础指标。

### 1.2 核心目标

| 目标 | 说明 |
|------|------|
| 实时行情 | 展示价格、涨跌额、涨跌幅、开盘/最高/最低价 |
| 量价数据 | 成交量和成交额 |
| 估值指标 | 市盈率(PE)、市净率(PB)、股息率(DY) |
| 行业筛选 | 按行业分类浏览自选股 |
| 日期选择 | 查看历史日期的行情快照 |
| ⓘ提示 | 每个字段带悬停说明工具提示 |

## 3. 业务逻辑

### 3.1 行情卡片

每只自选股渲染为一个卡片，包含：

| 字段 | 数据源 | 格式 |
|------|--------|------|
| 股票名称/代码 | watchlist | 名称 + 6位代码 |
| 价格 | quotes.price | 保留2位小数 |
| 涨跌额 | quotes.change | 带正负号 |
| 涨跌幅 | quotes.change_pct | 百分比 |
| 开盘/最高/最低 | quotes.open/high/low | 保留2位小数 |
| 成交量 | quotes.volume | 格式化(万手) |
| 成交额 | quotes.amount | 格式化(亿元) |
| PE | quotes.pe | 保留2位小数 |
| PB | quotes.pb | 保留2位小数 |
| 股息率 | quotes.dy | 百分比 |

### 3.2 ⓘ 工具提示

```javascript
// 每个字段旁的悬停提示
const hints = {

    dy: '股息率 = 每股股息 / 股价 × 100%',
    volume: '当日成交量，单位为手（1手=100股）',
    amount: '当日成交总金额'
}
```

## 4. 交互流程

```
挂载 → useDataStore 提供 quotes 数据
  → useIndustryStore 提供行业分类
  → 默认"全部"显示所有自选股的行情卡片
  → 切换行业 → 过滤显示该行业股票
  → 切换日期 → 查询历史行情快照
```
