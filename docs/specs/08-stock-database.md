# 模块8: 股票数据库构建

> **核心文件**: `scripts/build_stock_db.py` | **输出**: `data/a_stocks.json` (327KB)

---

## 1. 功能概述

合并沪深两市 A 股列表，生成全量股票搜索索引。包含中文→拼音映射以支持拼音搜索。为一次性执行脚本，仅在 A 股列表更新时重新运行。

---

## 2. 核心业务逻辑

### 2.1 数据源

| 来源 | 文件 | 代码列 | 名称列 |
|------|------|--------|--------|
| 深市 | `A股列表.xlsx` | 第 5 列 (iloc[4]) | 第 6 列 (iloc[5]) |
| 沪市 | `上A股列表.xls` | 第 1 列 (iloc[0]) | 第 3 列 (iloc[2]) |

### 2.2 拼音映射

内置 600+ 汉字→拼音的映射表 `py = {'一':'y', '二':'e', ...}`，覆盖 A 股常见股票名称用字。

```python
def gen_py(name: str) -> str:
    # 预处理：去掉全角空格、替换全角字母
    name = name.replace('\u3000','').replace(' ','').replace('Ａ','A').replace('Ｂ','B')
    # 逐字转换，非汉字保留原字符
    return ''.join(py.get(c, c if ord(c) < 128 else '') for c in name).lower()
```

### 2.3 去重

深市和沪市代码全部放入 `all_stocks[code]` 字典，同代码自动覆盖。

### 2.4 输出格式

```json
[
  {"code": "000001", "name": "平安银行", "market": "sz", "py": "payx"},
  {"code": "600000", "name": "浦发银行", "market": "sh", "py": "pfyx"},
  ...
]
```

## 5. 异常处理机制

| 场景 | 处理 |
|------|------|
| 代码非数字 | `code.isdigit()` 过滤 |
| 名称为空 | `if name:` 过滤 |
| 全角字符/空格 | `gen_py()` 预处理阶段归一化 |
